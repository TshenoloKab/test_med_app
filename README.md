# Medical Appointment Booking Website

## Overview
This is a frontend project for a Medical Appointment Booking system. 
Users can browse doctors, book appointments, and manage their bookings. 
This project demonstrates usage of React, Redux, and modern frontend practices.

## Features
- Browse available doctors and their specialties
- Book appointments for selected doctors
- Manage booked appointments
- Responsive and user-friendly interface
